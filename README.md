# Imporlan - Web Platform

Plataforma web completa para Imporlan, incluyendo landing page, panel de usuario y panel de administración.

## Estructura del Proyecto

```
imporlan/
├── index.html          # Landing page principal
├── admin/              # Panel de administración
├── api/                # Backend API (PHP)
├── panel/              # Panel de usuario (producción)
├── panel-test/         # Panel de usuario (pruebas)
├── test/               # Ambiente de pruebas completo
├── assets/             # Recursos estáticos (CSS, JS)
└── images/             # Imágenes del sitio
```
